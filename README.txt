-> Download Project
-> Type "npm install" and hit enter to install the node modules
-> Open the "json_server" location in preffered CMD
-> Type "json-server --watch db.json --host <YOUR COMPUTER IP ADDRESS> -p 3001 -d 2000" and hit enter to launch the json server
-> Open the project location in preffered CMD
-> Type "yarn start" and hit enter
